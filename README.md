# Obsidian Themed Site

## Purpose
For an assignment introducing GitHub pages in CTEC 127 PHP with SQL.
This was supposed to be just a simple static web page but I wanted to add some style fast with the help of Claude Sonnet 4.5.
I used Ai, more for speed sake and debugging practice. 

## Code
The notes are in JSON format, then parsed with JavaScript. 
The bottom floating action button with radial petals is straight CSS on Chrome based browsers, else its JavaScript. 
Inspired by watching Dylan Beattie's talk from NDC London 2026 om YouTube.
